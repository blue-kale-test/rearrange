Rearrange
=======

This module is used for rearranging names.
Turns "LastName, FirstName" into "FirstName, LastName".

## Example
* Calling rearrange _name ("Turing, Alan") will return '"Alan Turing"' I
* Calling rearrange _name ("Turing, Alan") will return '"Alan Turing"' I
* Calling rearrange _name ("Turing, Alan") will return '"Alan Turing"' I
